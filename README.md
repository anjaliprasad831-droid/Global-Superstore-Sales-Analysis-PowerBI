# Global-Superstore-Sales-Analysis-PowerBI
Data Analysis &amp; Power BI project featuring data preprocessing, exploratory data analysis (EDA), DAX, interactive dashboards, KPI reporting, and business insights using the Global Superstore dataset.

## Project Overview

This project analyzes the Global Superstore retail dataset to uncover sales trends, profitability patterns, customer behavior, and regional performance. The analysis was performed using Python for data cleaning and exploratory data analysis (EDA), while Power BI was used to create interactive business intelligence dashboards.

The project demonstrates end-to-end data analysis skills, including data preprocessing, visualization, business insights generation, and dashboard development.


## Objectives

- Analyze sales and profit performance across markets and regions.
- Identify top-performing products and customer segments.
- Evaluate the impact of discounts on profitability.
- Discover sales trends over time.
- Build interactive Power BI dashboards for decision-making.
- Generate actionable business recommendations.

## Dataset

Dataset: Global Superstore

The dataset contains over 50,000 retail transactions across multiple countries, markets, product categories, and customer segments.

Key fields include:

- Sales
- Profit
- Quantity
- Discount
- Category
- Sub-Category
- Customer Information
- Market
- Region
- Country
- Order Date
- Ship Date

## Key Insights

- Technology generated the highest sales and profit.
- Some products generated high revenue but low profit margins.
- Certain regions consistently outperformed others.
- High discounts did not always increase profitability.
- A small percentage of customers contributed significantly to total sales.
- Standard Class was the most frequently used shipping method.


## Business Recommendations

- Focus marketing efforts on high-profit product categories.
- Optimize discount strategies to improve profitability.
- Expand operations in high-performing markets.
- Improve inventory planning for top-selling products.
- Monitor underperforming regions for growth opportunities.
